# AI Experimentation Repository

Personal repository for exploring and experimenting with cutting-edge AI technologies.

## Projects

### [google_adk_agent](./google_adk_agent/)
Multi-agent system built with Google's Agent Development Kit (ADK). Features coordinated agents for YouTube Shorts content creation including scriptwriting, visual generation, and video editing workflows.

### [natural_lang_to_sql](./natural_lang_to_sql/)
Text-to-SQL conversion experiments using LangChain and LLMs. Enables natural language queries against SQL databases.

### [rag_test](./rag_test/)
Retrieval-Augmented Generation implementations exploring document QA, knowledge retrieval, and hybrid RAG architectures.

### [MCP](./MCP/)
Model Context Protocol server implementations for connecting AI models with external tools and data sources.

### [langchain_multi_agent](./langchain_multi_agent/)
Multi-agent orchestration experiments using LangChain, exploring agent collaboration and task decomposition.

## Tech Stack

- **LLMs**: OpenAI, Anthropic, Ollama
- **Frameworks**: LangChain, Google ADK, LangGraph
- **RAG**: Chroma, Pinecone, vector databases
- **Tools**: MCP, function calling, agentic workflows
